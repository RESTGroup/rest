Generate even-tempered auxiliary Gaussian basis (ETB).
This work references df module of PySCF.

Structs

InfoV2
This structure works to hold results generated by etb_gen_for_atom_list. It includes an hashmap named ‘elements’ where the keys are element names and the values are structures named crate::basis_io::Basis4Elem which stores basis set information for certain elements.
